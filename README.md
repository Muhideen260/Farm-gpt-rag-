# Farm-GPT 🌾

**Your AI Farming Assistant for Nigeria**

Farm-GPT helps Nigerian farmers get instant advice on:
- What crops to plant based on location and season
- Soil and pest tips
- Rainy vs Dry season planting

Built with Python + Streamlit

**Live Demo**: [Click here](https://Farm-gpt-rag.streamlit.app)

**Example**: Ask "Best crops to plant in Ibadan during rainy season"
